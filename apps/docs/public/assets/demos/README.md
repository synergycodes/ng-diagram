# Template Screenshots

Place template screenshots in this directory.

## Guidelines

- **Format**: PNG or JPG
- **Aspect Ratio**: 16:9 (recommended)
- **Recommended Size**: 800x450px
- **Naming**: Use kebab-case (e.g., `org-chart.png`)

## Current Screenshots

- `org-chart.png` - Screenshot for the Org Chart starter kit

To add a new template screenshot:

1. Save the image file in this directory
2. Add the corresponding template object to `/apps/docs/src/content/docs/templates/templates.ts`

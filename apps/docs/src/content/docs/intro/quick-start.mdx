Get up and running with ngDiagram in minutes. This guide will help you create your first interactive diagram.

## Installation

```bash
npm install ng-diagram
```

**Note**: You can use `yarn` or `pnpm` too

## Import Styles

**Important:** You must import the required styles for the diagram to display correctly.

```css
@import 'ng-diagram/styles.css';
```

Without these styles, the diagram components will not render properly.

## Create Your First Diagram

```typescript
import { Component } from '@angular/core';
import { NgDiagramComponent, NgDiagramContextComponent, initializeModel } from 'ng-diagram';

@Component({
  imports: [NgDiagramContextComponent, NgDiagramComponent],
  template: `
    <ng-diagram-context>
      <ng-diagram [model]="model" />
    </ng-diagram-context>
  `,
  styles: `
    :host {
      flex: 1;
      display: flex;
      height: 100%;
    }
  `,
})
export class AppComponent {
  model = initializeModel({
    nodes: [
      { id: '1', position: { x: 100, y: 150 }, data: { label: 'Node 1' } },
      { id: '2', position: { x: 400, y: 150 }, data: { label: 'Node 2' } },
    ],
    edges: [
      {
        id: '1',
        source: '1',
        sourcePort: 'port-right',
        targetPort: 'port-left',
        target: '2',
        data: {},
      },
    ],
  });
}
```

**Important:** The `<ng-diagram>` component must be wrapped with the `<ng-diagram-context>` component, which serves as a dependency injection container for diagram-specific services and state management.

That's it! You now have a working diagram with default node and edge templates.

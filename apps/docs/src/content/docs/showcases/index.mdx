## Submit your project

Built something with ngDiagram? We would love to feature it. Submissions are made by opening a pull request, so anyone can contribute.

Full instructions and the entry format live in the [showcases folder on GitHub](https://github.com/synergycodes/ng-diagram/tree/main/apps/docs/src/content/docs/showcases). The ngDiagram team reviews each submission and may decline entries that are spam or contain inappropriate content.

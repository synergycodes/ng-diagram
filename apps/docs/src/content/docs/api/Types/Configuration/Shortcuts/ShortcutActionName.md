> **ShortcutActionName** = [`KeyboardActionName`](/docs/api/types/configuration/shortcuts/keyboardactionname/) \| [`PointerOnlyActionName`](/docs/api/types/configuration/shortcuts/pointeronlyactionname/) \| [`WheelOnlyActionName`](/docs/api/types/configuration/shortcuts/wheelonlyactionname/)

All valid action names for shortcuts

Includes:
- Keyboard actions (e.g., 'keyboardMoveSelectionUp', 'copy', 'selectAll')
- Pointer-only actions (e.g., 'multiSelection', 'boxSelection')
- Wheel-only actions (e.g. 'zoom')

Configuration for zooming behavior.

## Properties

### max

> **max**: `number`

The maximum allowed zoom level.

#### Default

```ts
10.0
```

***

### min

> **min**: `number`

The minimum allowed zoom level.

#### Default

```ts
0.1
```

***

### step

> **step**: `number`

The zoom step increment.

#### Default

```ts
0.03
```

***

### zoomToFit

> **zoomToFit**: `ZoomToFitConfig`

Configuration for zoom-to-fit operations.

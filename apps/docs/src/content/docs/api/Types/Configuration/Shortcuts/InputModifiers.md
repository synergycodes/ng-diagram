## Properties

### meta

> **meta**: `boolean`

***

### primary

> **primary**: `boolean`

Ctrl key (Windows/Linux) OR Cmd key (Mac)

***

### secondary

> **secondary**: `boolean`

Alt key

***

### shift

> **shift**: `boolean`

Shift key

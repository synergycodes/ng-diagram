Configuration for the diagram background.

## Properties

### default

> **default**: `"grid"` \| `"dots"`

The default background pattern.

#### Default

```ts
'dots'
```

***

### dotSize?

> `optional` **dotSize**: `number`

The size of the dots pattern.

#### Default

```ts
60
```

***

### gridSize?

> `optional` **gridSize**: `number`

The size of the grid pattern.

#### Default

```ts
100
```

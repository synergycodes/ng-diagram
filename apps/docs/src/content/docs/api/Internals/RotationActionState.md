State tracking a node rotation operation in progress.

## Properties

### initialNodeAngle

> **initialNodeAngle**: `number`

Initial angle of the node before rotation.

***

### nodeId

> **nodeId**: `string`

ID of the node being rotated.

***

### startAngle

> **startAngle**: `number`

Angle in degrees at the start of the rotation operation.

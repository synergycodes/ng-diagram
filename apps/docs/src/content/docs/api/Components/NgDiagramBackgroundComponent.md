The `NgDiagramBackgroundComponent` is responsible for rendering the background of the diagram.

## Example usage
```html
<ng-diagram ... >
  <ng-diagram-background>
    <!-- Optional: custom SVG, HTML or IMAGE for background -->
  </ng-diagram-background>
</ng-diagram>
```

## Implements

- `AfterContentInit`

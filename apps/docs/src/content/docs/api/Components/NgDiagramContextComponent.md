The `NgDiagramContextComponent` sets up the dependency injection context for all ngDiagram services.
It should wrap any usage of [NgDiagramComponent](/docs/api/components/ngdiagramcomponent/).

## Example usage
```html
<ng-diagram-context>
  <ng-diagram ... />
</ng-diagram-context>
```
